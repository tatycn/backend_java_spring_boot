## Environment Prepare

Install openjdk 11

```bash
apt install openjdk-11-jdk-headless
```

Setting JAVA_HOME:

```bash
vi ~/.bash_profile
```
Add

```bash
export JAVA_HOME="/usr/lib/jvm/java-11-openjdk-amd64/"
```
Activate

```bash
source ~/.bash_profile
```
